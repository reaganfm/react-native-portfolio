Code Examples


Menu App

This menu app uses React to show multiple food choices in a list each with a button to add to your cart. This was made to get started to using React Native and its functions.

Task List

This is a task list app that provides an interface for users to add and complete tasks. At the bottom you can type in a new task and add it to the list on the bottom right. You can mark a task as complete by clicking on the checkbox next to the task and it will strikethrough the task.

Exercise App

This is an exercise app that contains three different types of exercises. It includes swimming, which has a stopwatch function, push-ups, which has a counter function, and running, which has a stopwatch function that includes a way to record laps.

Quiz App

This is a quiz app that has three different types of questions: multiple choice, true or false, and multiple answer. After answering the questions, your results are displayed showing your answers and what the correct answers were.

Figma Apps


Exercise App

This Figma design is a thought out exercise app which includes three exercises: push-ups, running, and planks. The running exercise was expanded upon to include tracking and saving past runs.

To-Do App

This Figma design is a to-do list app which contains a log in screen, sign out screen, and to-do list functions. These functions include: creating a new task, marking a task as complete, editing a task, filtering by completion, and deleting tasks.

Flash Card App

This Figma design is a flash card study app which shows a list of flashcards that you can open and see the terms, definitions, and study them. You have the option to make a new flashcard set or edit one. When studying a set, you can reveal the answer, go forward or back in the list, and shuffle the cards.
